# Анализ поведения клиентов и транзакций (Зачетная работа по SQL)

В этом репозитории содержится финальная практическая работа, посвященная анализу данных о клиентах и их покупках за годовой период (с июня 2015 по июнь 2016 года). 

## Исходные данные:
* `customers_final.csv` — профили клиентов (пол, возраст и уникальный ID).
* `transactions_final.csv` — история транзакций (даты, ID чеков, суммы оплат и ID клиентов).

## Какие задачи были решены:

1. **Анализ лояльности и регулярности**: Выделена группа постоянных клиентов с непрерывной историей покупок (без пропусков месяцев в течение года). Для них рассчитаны средний чек, количество операций и средние траты за месяц.
2. **Ежемесячная бизнес-аналитика**: Написаны запросы для расчета динамики среднего чека, активности пользователей (MAU), а также долей операций и выручки каждого месяца в рамках года. Дополнительно посчитано процентное соотношение клиентов по полу (M/F/NA) и их доли в общих затратах.
3. **Сегментация по возрастным группам**: Клиенты распределены по когортам с шагом в 10 лет. Написан комплексный аналитический запрос с использованием оконных функций для расчета поквартальной динамики среднего чека, активности и долей затрат для каждой возрастной группы.

## 🛠️ Какие навыки и функции SQL использовались:
* Сложные подзапросы и обобщенные табличные выражения (`WITH ... AS`).
* Агрегатные функции с условной логикой (`CASE WHEN`).
* Оконные функции (`SUM() OVER(PARTITION BY...)`) для расчета долей рынка и процентов.
* Объединение таблиц (`JOIN`, `LEFT JOIN`, `USING`) и группировка данных.
